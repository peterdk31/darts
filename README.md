# Dart Game Tracker

A static, single-page React webapp for scoring 501, 301, Cricket, and Around the Clock at a physical dartboard. Mobile-first, offline-capable, deployable as static assets to GitHub Pages.

See [`specs/001-dart-game-tracker/quickstart.md`](./specs/001-dart-game-tracker/quickstart.md) for the manual test walkthrough and dev-setup steps.
